# A ligthweight Neovim rice that does pretty much all you need.

**NOTE:** NPM and [Nerdfont](https://www.nerdfonts.com/) is required.

A template for getting started with [AstroNvim](https://github.com/AstroNvim/AstroNvim)

## üõ†Ô∏è Installation

#### Make a backup of your current nvim and shared folder

```shell
mv ~/.config/nvim ~/.config/nvim.bak
mv ~/.local/share/nvim ~/.local/share/nvim.bak
mv ~/.local/state/nvim ~/.local/state/nvim.bak
mv ~/.cache/nvim ~/.cache/nvim.bak
```

#### Clone the repository

```shell
git clone https://github.com/fspamo/nvim ~/.config/nvim
```

#### Start Neovim

```shell
nvim
```

**NOTE:** Couple optional keybinds are assigned to different key combinations. (Such as Space+S to save the file and Space+F to format)
